Post Types Carousel & Slider is simple way to build slider and carousel.

== Description ==
By using this plugin you can display posts OR custom posts as a slider OR carousel. Next and previous post data fetch by Ajax.

Demo : [http://aeonian.in/post-cs/](http://aeonian.in/post-cs/)

**How to create new slider or carousel and use design options**

https://www.youtube.com/watch?v=kk336decgLw

== Features ==
01. Unlimited number of sliders and carousels
02. Retrieves posts by Post Types
03. Ascending or descending order and Sort retrieved posts
04. Show posts associated with certain categories, taxonomies & tags
05. Show posts based on a keyword search
06. Update HTML template from plugin option
07. CSS animation (https://daneden.github.io/animate.css/)
08. Multiple design option
09. Responsive
10. Admin preview

== Installation ==

= Admin Installer via search =
1. Visit the Add New plugin screen and search for "post types carousel slider".
2. Click the "Install Now" button.
3. Activate the plugin.
4. Navigate to the "Post Types Carousel & Sliders" Menu.

= Admin Installer via zip =
1. Visit the Add New plugin screen and click the "Upload Plugin" button.
2. Click the "Browse..." button and select zip file from your computer.
3. Click "Install Now" button.
4. Once done uploading, activate Post Types Carousel & Sliders.

= Manual =
1. Upload the Post Types Carousel & Sliders folder to the plugins directory in your WordPress installation.
2. Activate the plugin.
3. Navigate to the "Post Types Carousel & Sliders" Menu.
